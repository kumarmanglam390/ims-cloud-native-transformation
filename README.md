# IMS MRF Deployment

## Overview

Media Resource Function (MRF) deployment architecture for IMS environments.

The solution demonstrates deployment flow, call processing flow, media announcement handling and automation possibilities using DevOps tools.

## Components

- P-CSCF
- I-CSCF
- S-CSCF
- Application Server
- MRFC
- MRFP

## Features

- Media Announcements
- IVR Services
- Conferencing
- RTP Media Handling

## DevOps Integration

- Docker
- Kubernetes
- GitHub Actions
- CI/CD Pipeline
